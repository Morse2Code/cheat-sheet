HTML REFERENCE SHEET

Tags of the Table

Tag Name Description
table Table The wrapper element for all HTML tables.
thead Table Head The sets of rows defining the column headers in a table.
tbody Table Body The sets of rows contaning actual table data.
tr Table Row The table row container
td Table Data The table row container.
tfoot Table Foot The set of rows defining the footer ina table.

Table Attributes

Tag Name Description
colspan Col Span Defines howmany colums a td element should span.
rowsspan Rows Span Defines how many rows a td element should span.
